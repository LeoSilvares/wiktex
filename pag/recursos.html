Recursos multimída

  • \includegraphics[argumentos opcionais]{imagem}: insere uma imagem do repositório de mídia, que pode ser acessado no botão "Mídia" da página de edição. Os argumentos opcionais devem ser na forma

    [Opção1=valor, ..., OpçãoN=valor]

    onde, cada OpçãoX corresponde a um campo de estilo CSS ho html (exemplos: width [largura], height [altura])

  • \youtube{id do vídeo}{texto explicativo}: Insere uma caixa com texto explicativo que, quando clicada, abre um vídeo do YouTube. O id do vídeo são as letras/números que aparecem no final do link do vídeo. Exemplo de vídeo:

    Assista ao vídeo "Expressões do Primeiro Grau", que cobre parte do conteúdo desta página

    criado com o comando

    \youtube{JJ-Y65hFIrA}{Assista ao vídeo "Expressões do Primeiro Grau", que cobre parte do conteúdo desta página}

  • \geogebra[argumentos opcionais]{imagem}: insere uma imagem do repositório de mídia, que pode ser acessado no botão "Mídia" da página de edição. Os argumentos opcionais devem ser na forma

    [Opção1=valor, ..., OpçãoN=valor]

    onde, cada OpçãoX pode corresponder a a um campo de estilo CSS ho html (exemplos: width [largura], height [altura]), ou a um parâmetro do GeoGebra, se inicado com "G_" (por exemplo, "G_width=300" passará o parâmetro width como 300 para o applet).
