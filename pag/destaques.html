Ambientes de destaque no texto

  • Dica: o código 

                \begin{tip}
                Isto é uma dica.
            \end{tip}

    produzirá:

    Isto é uma dica.

  • Alerta: o código 

                \begin{alert}
                Isto é um alerta.
            \end{alert}

    produzirá:

    Isto é um alerta.

  • Ajuda/pergunta: o código 

                \begin{help}
                Isto é uma ajuda ou pergunta.
            \end{help}

    produzirá:

    Isto é uma ajuda ou pergunta.

  • Importante: o código 

                \begin{important}
                Isto é importante!
            \end{important}

    produzirá:

    Isto é importante!

  • Informação: o código 

                \begin{info}
                Isto é uma informação!
            \end{info}

    produzirá:

    Isto é uma informação!
