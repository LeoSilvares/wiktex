Ambientes de teoremas e definições

Os seguintes ambientes são suportados:

  • Definição: o código 

                \begin{definition}
                Isto é uma definição.
            \end{definition}

    produzirá:

    Definição 1

    Isto é uma definição.

  • Teorema: o código 

                \begin{theorem}
                Isto é um teorema.
            \end{theorem}

    produzirá:

    Teorema 1

    Isto é um teorema.

  • Proposição: o código 

                \begin{proposition}
                Isto é uma proposição.
            \end{proposition}

    produzirá:

    Proposição 1

    Isto é uma proposição.

  • Exemplo: o código 

                \begin{example}
                Isto é uma definição.
            \end{example}

    produzirá:

    Exemplo 1

    Isto é um exemplo.

  • Solução: produz uma área com solução de exemplos, que pode ser expandida. É especialmente útil dentro de exemplos, como abaixo: 

                \begin{example}
                Isto é um exemplo.
                \begin{solution}
                    Isto é uma solução.
                \end{solution}
            \end{example}

    produzirá:

    Exemplo 2

    Isto é um exemplo.

    Solução

    Isto é uma solução.

  • Demonstração: produz uma área para demonstração que pode ser expandida. 

                \begin{proof}
                Tome $x=...$
            \end{proof}

    produzirá:

    Demonstração

    Tome $x=...$
