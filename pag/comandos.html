Comandos suportados

Formatação

  • \textbf{texto}: formata o texto entre chaves com negrito.

  • \textit{texto}: formata o texto entre chaves com itálico.

  • \emph{texto}: formata o texto entre chaves com ênfase.

  • \underline{texto}: formata o texto entre chaves com sublinhado.

  • \texttt{texto}: formata o texto entre chaves com monoespaçado.

  • \hspace{espaço}: insere um espaço horizontal.

  • \vspace{espaço}: insere um espaço vertical.

  • \bigskip: pula uma linha.

Organização do documento e navegação

  • \section{Nome da seção}: cria uma nova seção

  • \subsection{Nome da subseção}: cria uma nova subseção

  • \label{rótulo}: insere um rótulo para referência. No código abaixo, o comando \label{pitagoras} atribui o rótulo pitagoras ao teorema. O comando \ ref{pitagoras} exibe o número do teorema. 

        \begin{theorem}\label{pitagoras}
        Se $ABC$ é um triângulo retângulo com ângulo reto $\hat{A}$, então $\overline{BC}^2=\overline{AB}^2+\overline{AC}^2$.
    \end{theorem}
    
    \[\vdots\]
    
    Pelo Teorema \ref{pitagoras}, temos que ...

    A saída do código acima é:

    Teorema 1

    Se $ABC$ é um triângulo retângulo com ângulo reto $\hat{A}$, então $\overline{BC}^2=\overline{AB}^2+\overline{AC}^2$.

    \[\vdots\]

    Pelo Teorema 1, temos que ...

  • \ ref{rótulo}: insere um link para um rótulo criado com \label, como explicado no comando \label.

  • \href{url}{texto}: insere um link para o endereço informado com o texto dado.
