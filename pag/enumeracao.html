Ambientes de enumeração

  • Enumeração padrão: o código 

            \begin{enumerate}
            \item Item
            \item Outro item
        \end{enumerate}

    produzirá:

  1. Item

  2. Outro item

  • Enumeração com estilo: o código 

            \begin{enumerate}[(a)]
            \item Item
            \item Outro item
        \end{enumerate}

    produzirá:

  1. Item

  2. Outro item

Pode-se utilizar no estilo os caracteres '(', ')', '-', '.' e ':'. A numeração pode ser com 1 (números), I (romanos maiúsculos), i (romanos minúsculos), a (letras latinas minúsculas) ou A (letras latinas maiúsculas).

  • Itemização com estilo: o código 

            \begin{itemize}
            \item Item
            \item Outro item
        \end{enumerate}

    produzirá:

$\;$

  • Item

  • Outro item

  • Itemização com colunas: o código 

            \begin{itemize-cols}{200px}
            \item Item 1
            \item Item 2
            \item Item 3
            \item Item 4
            \item Item 5
            \item Item 6
        \end{itemize-cols}

    produzirá uma lista em que os elementos podem ficar lado a lado, em caixas de 200px de largura, que se distribuirão de acordo com a largura da página:

  • Item 1

  • Item 2

  • Item 3

  • Item 4

  • Item 5

  • Item 6

Redimensione a janela do navegador para ver o efeito.
