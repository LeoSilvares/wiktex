\section{Guia de edição}

Este guia apresenta os comandos do LaTeX já implementados e novos comandos, para suportar, por exemplo, recursos multimídia. Está dividi

\begin{itemize}
    \item \textbf{Comandos suportados:} comandos de formatação, seção e referências
    \item \textbf{Recursos multimídia:} comandos de inclusão de imagens, vídeos e GeoGebra
    \item \textbf{Modo matemático:} como usar o modo matemático
    \item \textbf{Ambientes de definições e teoremas:} ambientes de definições, resultados, exemplos
    \item \textbf{Ambientes de destaque:} caixas de informação, aviso, importante, etc.
    \item \textbf{Ambientes de enumeração:} enumerate e itemize
    \item \textbf{Ambientes de layout:} minipage e tabelas
    \item \textbf{Ambientes específicos:} ambiente de inclusão de código HTML
\begin{itemize}
